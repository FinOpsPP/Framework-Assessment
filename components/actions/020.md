# 20: Embed Reports In Workflows

**Specification Version:** 1.8.0

**Slug:** 20-embed-reports

**Implementation Types:**

* None

## Descriptive Statement

Embed reports into workflows (Slack/Jira/Teams) and set alerting.

## Supplemental Guidance

Scoring for this actions is treated as percent. That means that if you have
alerting in most tooling, for example, you have implicitly achieved it in some
toolings already.

## Scoring Guide

**Weight:** 1.0

### Formula

None

### Scoring Details

**Score Type:** percent

| Score | Condition |
| ----- | --------- |
| 0 | No alerting in workflow tools |
| 1 | Alerting in at least one tool |
| 3 | Alerting in some tooling [>20%] |
| 6 | Alerting in most tooling [>60%] |
| 10 | Alerting in all workflow tools [100%] |
