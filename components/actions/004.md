# 4: Define Data Granularity & Frequency

**Specification Version:** 1.8.0

**Slug:** 4-define-granularity

**Implementation Types:**

* None

## Descriptive Statement

Define data granularity; include mandatory metadata.

## Supplemental Guidance

Scoring on this action is treated as sequential. If you have granularity
of say 2 days, for example, you have implicitly achieved all previous levels
of granularity.

## Scoring Guide

**Weight:** 1.0

### Formula

None

### Scoring Details

**Score Type:** threshold

| Score | Condition |
| ----- | --------- |
| 0 | Never |
| 1 | Monthly or greater |
| 2 | More than a week |
| 3 | 7 days |
| 4 | 5 days |
| 6 | 2 days |
| 8 | 1 day |
| 10 | Instant |
