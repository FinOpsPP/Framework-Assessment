# 25: Automate Alert Routing

**Specification Version:** 1.8.0

**Status:** Accepted

**Slug:** 25-automate-alert

**Implementation Types:**

* None

## Descriptive Statement

Automate alert routing to owners via existing channels (email/Slack/Teams).

## Supplemental Guidance

Scoring for this actions is treated as percent. That means that if you have
alerting to most owners, for example, you have implicitly achieved it in some
owners already.

## Scoring Guide

**Weight:** 1.0

### Formula

None

### Scoring Details

**Score Type:** percent

| Score | Condition |
| ----- | --------- |
| 0 | No alerting to any owners |
| 1 | Alerting in at least one owner |
| 3 | Alerting to some owners [>20%] |
| 6 | Alerting to most owners [>60%] |
| 10 | Alerting to all owners [100%] |
