# 133: Coordinate Policy Changes Broadly

**Specification Version:** 0.8.0

**Slug:** 133-coordinate-policy

**Implementation Types:**

* None

## Descriptive Statement

Coordinate policy/gov changes cross-functionally; communicate broadly.

## Supplemental Guidance

Policies in this case could include changing existing workflows to include both teams, etc.
There may not always be new policies created as a result of intersecting disciplines.

## Scoring Guide

**Weight:** 1.0

### Formula

1. Document new policies as a result of cross-functional collaboration goals
2. Broad policy changes are accepted by governing and stakeholder teams
3. Communication is distributed by all governing teams using shared language
4. Change management procedures are followed for updated policy implementation

10*Ceil(x/4)
