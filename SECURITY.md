# Security Policy

## Supported Versions

Since the core specifications in this project could be considered Configuration-as-Code, generally there is not much to support that or the generated files markdown files that are created from them.

When it comes to the Python code and assessments, support is on a "best-effort" bases. Meaning only the most recent minor versions will typically be supported, and all previous minor versions will not

## Reporting a Vulnerability

For any relevant secret vulnerabilities that are found in the python code, please feel report them as a Bug Report issue, selecting the `tools` type. Inside the vulnerability description in the Bug Description textbox.
